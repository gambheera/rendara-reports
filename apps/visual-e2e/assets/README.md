# Vendored fonts

`inter-latin-400-normal.woff2` — **Inter**, Latin subset, weight 400. Vendored
from [`@fontsource/inter`](https://www.npmjs.com/package/@fontsource/inter) so
the visual-regression fixture embeds a pinned font and never depends on system
font availability.

Inter is licensed under the **SIL Open Font License 1.1**
(<https://github.com/rsms/inter/blob/master/LICENSE.txt>). It is used here only
as a test fixture; it is not bundled into any published package.
